# Conclusion

In this assignment, there are mutiple different cybersecurity tools used in both reconnaissance and maintaining access has been explored. Through hands on practice in a controlled lab environment, it give more understanding how tools like Nmap, DNSRecon, Hping3, and Recon-ng are used to gather information about a target system or vunerable system. Each tool has a different purpose and Some focus on scanning networks and finding open ports while others collect DNS information or public data. This shows that reconnaissance is not just focus on one activity but a combination of different techniques to understand a target system better.

It give exposure how maintaining access using tools such as PowerSploit, Webshells, Weevely, Dns2tcp, and Cryptcat work. These tools use different methods to keep access to a compromised system such as encrypted communication, system configuration changes or hidden network channels. By comparing them, we can understand that each tool is designed for different environments and situations.

Overall, this assignment helped us understand how attackers think and how different tools are used at each stage. At the same time, it also showed the importance of proper security controls and system monitoring. Learning how these tools work makes it easier and give some ideas to understand how to defend against attackers in real world situations.
